Sobre o Sistema

Este é um sistema de contingência (vendas offline), criado para dar suporte às filiais e equipes quando houver indisponibilidade do sistema oficial, internet ou energia.

Aqui é possível realizar simulações de vendas e organizar negociações de forma profissional, para posteriormente registrar tudo no sistema oficial da empresa.

Este sistema não possui fins lucrativos, sendo apenas uma ferramenta de apoio para garantir agilidade, transparência e continuidade no atendimento ao cliente.
